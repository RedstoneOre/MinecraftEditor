# Minecraft Editor
### Edit files just like playing minecraft
\(Cannot save yet\)

## Installation
This project can only be runned on Linux
```sh
git clone https://github.com/RedstoneOre/MinecraftEditor.git

```

## Usage
```sh
./editor <file>
```

## Operation

Use w,a,s,d to move

Use i,j,k,l to move digging focus

Use \[ to dig

Use ^C to Interrupt
